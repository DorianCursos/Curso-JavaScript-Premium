# Objeto Math

## ¿Para qué sirve?

Es un objeto incorporado en JavaScript que proporciona propiedades y métodos para realizar operaciones matemáticas y numéricas.

## Propiedades

La más utilizada es PI, para acceder a este valor simplemente tenemos que utilizar Math.PI, es un valor que ya establecido el objeto Math y que podemos utilizar para hacer cálculos geométricos.

## Métodos de redondeo

Math.abs(x): Devuelve el valor absoluto de x.
Math.ceil(x): Devuelve el entero más pequeño mayor o igual que x.
Math.floor(x): Devuelve el entero más grande menor o igual que x.
Math.round(x): Devuelve el valor redondeado de x al entero más cercano.

## Métodos de potencia y raiz cuadrada

Math.pow(base, exponente): Devuelve el valor de base elevado al exponente.
Math.sqrt(x): Devuelve la raíz cuadrada de x.

## Métodos de valores máximos y mínimos

Math.max(n1,n2,...): Devuelve el valor máximo entre los argumentos proporcionados.
Math.min(n1, n2, ...): Devuelve el valor mínimo entre los argumentos proporcionados.

## Métodos de generación aleatoria de números

Math.random(): Devuelve un número aleatorio entre 0 (inclusive) y 1 (exclusivo).
